# What do LLMs value? An evaluation framework for revealing subjective trade-offs in assessment of glycemic control


Code for manuscript presented at the Machine Learning for Health Symposium and published in the Proceedings of Machine Learning Research 


Overview: 
1. Perturbation experiments can be replicated using notebooks in perturbation_experiments/

2. run_all.ipynb runs the main experiments 

3. analyze_batch_run.ipynb produces main figures in the manuscript.

Note: the raw CGM data is not included in this repository
